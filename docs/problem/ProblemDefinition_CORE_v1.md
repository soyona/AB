# Problem Name

一年级学生课后生字学习持续性不足

---

# Problem Statement

一年级学生正处于识字能力快速发展的阶段，需要持续学习和巩固汉字。  
学校课堂能够提供基础教学，但课后用于生字巩固和练习的时间有限。  

在家庭场景中，学生并非单纯缺乏学习动机。观察显示，孩子希望掌握汉字，也会在写错后表现出改进意愿，但对重复、单调、以练习量为中心的方式更容易产生抵触。  

当孩子暂时无法掌握某个汉字时，较高的自我期待可能转化为挫败感，进而影响持续参与。  
家长虽然意识到生字学习的重要性，但依靠提醒、监督和要求的方式，很难长期维持孩子的积极体验和稳定参与。  

最终导致课后生字练习方式与孩子的掌握动机不匹配，练习频率不足，识字积累速度下降，阅读能力发展受到影响。

---

# Target User

## Primary User
一年级学生（6~7岁）

特点：
- 刚开始系统学习汉字
- 注意力持续时间有限
- 更偏好娱乐活动
- 希望掌握汉字并对自己有较高期待
- 对重复练习更容易产生抵触
- 未能掌握时可能出现挫败感
- 需要即时反馈和成就感

---

## Secondary User
学生家长

特点：
- 希望帮助孩子提高识字能力
- 希望培养阅读习惯
- 缺少有效的课后练习方式
- 难以长期监督和陪伴练习

---

# Current Situation

当前生字学习主要依赖：
- 学校课堂教学
- 家庭作业
- 家长督促阅读
- 课后重复抄写练习

课后时间中：
- 孩子更愿意玩耍
- 阅读持续时间较短
- 生字练习容易中断
- 重复练习容易引发抵触
- 掌握动机存在，但练习方式与学习风格不匹配

---

# Evidence

## Observation-1
学生每天放学后更愿意进行娱乐活动。

## Observation-2
学生对主动练习生字缺乏兴趣。

## Observation-3
每天20分钟阅读目标难以持续坚持。

## Observation-4
学校课堂内容不足以覆盖长期识字积累需求。

## Observation-5
家长需要持续提醒和督促。

## Observation-6
The student shows willingness to improve after mistakes and may voluntarily repeat incorrect characters.

This suggests mastery motivation exists.

## Observation-7
Resistance appears stronger toward repetitive practice than toward literacy learning itself.

This suggests a mismatch between learning style and practice method.

---

# Impact

## Student Impact
- 生字积累速度下降
- 偏旁部首掌握不足
- 汉字读音记忆不牢固
- 组词能力发展缓慢
- 阅读能力提升速度下降

## Parent Impact
- 陪伴成本增加
- 督促压力增加
- 家庭学习冲突增加
- 学习效果难以保证

---

# Existing Workarounds

目前主要采用：
- 学校布置作业
- 家长提醒
- 强制阅读
- 抄写练习
- 纸质练习册

存在问题：
- 趣味性不足
- 持续性较差
- 依赖家长监督
- 容易把掌握动机转化为挫败感
- 重复练习方式与孩子学习风格不匹配

---

# Problem Boundary

## In Scope
- 生字学习持续性不足
- 课后练习参与度不足
- 阅读习惯难以坚持
- 生字巩固效率不足
- 掌握动机与重复练习方式之间的不匹配
- 未能掌握汉字时产生的挫败感

## Out Of Scope
- 学校课程改革
- 教材设计
- 全学科辅导
- 数学学习
- 英语学习
- 长篇阅读能力培养

---

# Success Criteria

当问题改善时，应观察到：

1. 学生更愿意进行课后生字练习  
2. 学生能够持续参与生字学习  
3. 生字练习频率提升  
4. 阅读坚持时间提升  
5. 家长监督成本下降  
6. 生字掌握情况改善  
7. 学生的掌握动机得到保护，而不是被重复练习消耗  

---

# Constraints

- 面向一年级学生  
- 课后家庭场景  
- 不依赖学校教学改革  
- 不增加家长额外教学负担  

---

# Assumptions

- 学生具备基础识字能力  
- 家长愿意提供必要支持  
- 课后存在可用于练习的时间
